# PhaserQuasarVuePWA

Scaffolding for a PWA using Quasar, Vue, PhaserJS and Webpack

> Playful Platypus LLC

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ quasar dev

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```
